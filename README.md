# French Trains Performance Analysis (Python)

## Overview
This project analyzes the performance of the French railway network using the “French Trains” dataset" collected by SNCF.
The analysis focuses on understanding the main factors impacting train delays, cancellations and overall service performance.

## Objectives
The main objectives of this project are to:
- Analyze train performance indicators over time
- Identify the most frequent causes of delays and cancellations
- Explore demand and delay patterns using data analysis
- Provide data-driven insights to support performance improvement

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Interpretation of results from a business perspective
- Machine learning techniques for pattern identification

## Disclaimer
This project is for academic and portfolio purposes only and does not represent official SNCF internal analysis.
